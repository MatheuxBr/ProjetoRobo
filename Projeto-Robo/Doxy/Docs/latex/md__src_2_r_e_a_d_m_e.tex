Este projeto tem como objetivo o desenvolvimento de um robô controlado por microcontrolador Atmega328p, utilizando sensores e atuadores, como motor DC, sensor infravermelho (VS1838), LDR (fotoresistor), e um laser vermelho. O robô é capaz de realizar movimentos, direcionar o laser, efetuar disparos e executar outros comportamentos definidos no edital.\hypertarget{md__src_2_r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{\texorpdfstring{Participantes\+:}{Participantes\+:}}\label{md__src_2_r_e_a_d_m_e_autotoc_md2}
